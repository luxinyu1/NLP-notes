> 网站需要加载外网 js 脚本和访问 github 资源，可能部分地区会出现加载缓慢 / 显示不全的问题，可以尝试在使用 VPN 的环境下访问 🚀

## How to contribute?

1. 🧑‍🤝‍🧑 Fork 本项目
2. ✏️ 在 ```docs/```对应的目录下添加 Markdown 笔记
3. ✨ 更新受影响的目录
4. ✔️ 本地运行，检查笔记是否显示正常
    ```shell
    npx docsify serve docs
    ```
5. 🎈 提交 Pull Request
